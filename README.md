# StrangerThings
Elaboração de um projeto no âmbito da disciplina de Desenvolvimento de Aplicações Web I (DAW I), que consiste na criação de um site utilizando a estrutura Bootstrap, abordando o universo da renomada série "Stranger Things".

#### Link: https://SamukaCode.github.io/StrangerThings
Nota: 10,0.

## Stack utilizada

**Front-end:** HTML, CSS, BootStrap
